# Social Media Web Application

This is a modern social media web application built using **React** and **Vite**, focused on demonstrating core social platform features such as authentication, stories, posts, messaging UI, user discovery, connections, and profile management. The application uses **Clerk** for authentication and **Tailwind CSS** for responsive UI design.

---

## Features

- User authentication and session management using Clerk
- Stories (text, image, and video)
- Story viewer with progress indicator
- Feed with posts and media support
- Like, comment, and share UI
- Messaging interface (frontend only)
- User discovery and search
- Connections management (followers, following, pending)
- User profiles with edit functionality
- Responsive design with Tailwind CSS

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Clerk Authentication
- React Router DOM
- Moment.js
- Lucide React Icons
